MediaPlayer

The custom media player library. Unify the player UI that differs across browsers.

Download: GitHub

Usage

1. Load the stylesheet and javascript.

HTML
<link rel="stylesheet" href="mediaplayer.css">
<script src="mediaplayer.js"></script>

2. Add audio/video element.

HTML
<audio src="sample.mp3"></audio>
<video src="sample.mp4"></video>

3. Create objects of MediaPlayer class after loading the DOM.

JavaScript
window.addEventListener('DOMContentLoaded', () => {
document.querySelectorAll('audio').forEach(e => new MediaPlayer(e, 'audio'));
document.querySelectorAll('video').forEach(e => new MediaPlayer(e, 'video'));
});

Custom

You can easily change the color via CSS custom properties.

Audio

CSS
.mediaplayer-block[data-type="audio"] {
--mediaplayer-maincolor: hsl(0 0 30);
--mediaplayer-backcolor: hsl(0 0 90);
--mediaplayer-seekcolor1: hsl(0 0 30);
--mediaplayer-seekcolor2: hsl(0 0 60);
}

Video

CSS
.mediaplayer-block[data-type="video"] {
--mediaplayer-maincolor: hsl(0 0 30);
--mediaplayer-backcolor: hsl(0 0 90);
--mediaplayer-seekcolor1: hsl(0 0 30);
--mediaplayer-seekcolor2: hsl(0 0 60);
}
